# Finance_tracker:
Aplikace v PyQt5, která zobrazuje příjmy a výdaje přihlášenému uživateli. Částky je schopna screenshotovat pomocí easyOCR.

# Spuštění aplikace:
- pip install -r requirements.txt
- python src/main.py
- vytvoření účtu a přihlášení
